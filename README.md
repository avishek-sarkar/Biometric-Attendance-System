# Biometric Attendance System

A Laravel-based system for managing attendance using biometric verification.

## Prerequisites

- PHP CLI version 7.4.3
- Composer version 2.8.5
- MySQL/MariaDB
- Git
- Node.js & NPM (for asset compilation)
- Apache/Nginx web server

## Installation Steps

1. Clone the repository from the this branch
2. If anything is needed, please install it
